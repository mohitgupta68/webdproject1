This is a breaf introduction of some usefull HTML tags:-

  • <p>- used for writing paragraphs.
  • <h1> to <h6>- used for writing Headings each one is smaller from previous one.
  • <img src="">- used for adding images.
  • <a href="">- used for adding links.
  • <button>- used for adding buttons.
  • <br>- used for line break.
  • <hr>- used for horizontal line.
  • <abbr>- used for abbrivation.
  • <b>- used for bold text;
  • <cite>- used for citation.
  • <del>- used for deleting a word.
  • <em>- used for emphasized text.
  • <i>- used for itallic text.
  • <font>- used for specifying font size, face, family, color.
  • <iframe src="">- used for specifying inline frame and opening awebsite within itself.
  • <mark>- used for highlighting parts of a text.
  • <ol>- used for ordered list.
  • <ul>- used for unordered list.
  • <li>- used for list content.
  • <pre>- used for preformated text.
  • <q>- used for marking quotation.
  • <sub>- used for writing subscript.
  • <sup>- used for writing superscript.
  • <small>- used for writting smaller text.
  • <span>- used to color a part of text.
  • <strong>- it defines the text is important.
  • <u>- used for underlining a word.
  • <table>- used for inserting a table
  • <thead>- used for table head.
  • <th>- used for table head content.
  • <tbody>- used for table body.
  • <tr>- used for row wise insertion of table data.
  • <td>- used for inserting table data.
  • <tfoot>- used for table foot.
